# Welcome to MDX in Next.js!

This is a demo of using **MDX** with the Next.js App Router.

![Alt text](/me.jpg 'title')

## Features

- Write Markdown _and_ JSX in one file
- Use Next.js components (like `Image` above)
- Custom components via `mdx-components.tsx`
- Supports remark/rehype plugins (e.g. GFM tables)

Enjoy writing content with [MDX](https://mdxjs.com 'title')!

For more info, see the [Next.js MDX documentation](https://nextjs.org/docs/app/guides/mdx).
